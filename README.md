# Random User Explorer

Taller Práctico — Curso de NestJs  
Consumo de API Pública con TypeScript + Vite

## ¿Cómo correrlo?

```bash
npm install
npm run dev
```

Luego abre http://localhost:5173 en tu navegador.

## Estructura del proyecto

```
src/
├── types.ts    # Interfaces TypeScript para la API
├── api.ts      # Funciones de consumo (async/await + try/catch)
├── app.ts      # Lógica de la interfaz
├── styles.css  # Estilos
└── main.ts     # Punto de entrada
```

## API utilizada

[Random User API](https://randomuser.me) — genera datos ficticios de usuarios.
